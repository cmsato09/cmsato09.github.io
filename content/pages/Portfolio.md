## Dragon Quest Monsters Database (FastAPI+Streamlit)
This is the game information database for the Gameboy game called 'Dragon Warrior Monsters,' also known in Japan as 'Dragon Quest Monsters -- Terry's Wonderland. '
It is a read-only database using SQLite modeled by FastAPI in the back-end and Streamlit for the front-end. 

[FastAPI back-end Github repo](https://github.com/cmsato09/DQMonstersDB-API)

[Streamlit front-end Github repo](https://github.com/cmsato09/DQMonstersDB-streamlit)

FastAPI deployed on Deta. Access the SwaggerUI docs at  <https://em4lp8.deta.dev/docs>

Streamlit deployed on Streamlit Cloud. Access at  <https://dqmonstersdb.streamlit.app>
