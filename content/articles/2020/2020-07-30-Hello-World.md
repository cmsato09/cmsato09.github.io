## Post #1 My first post
This is my first blog post. Ever. 
I have little programming background and I decided to start learning how to program in the midst of a Covid 19 pandemic and small quarter-life crisis.

This blog will follow how I learn programming and serves to show how I develop as a programmer (as long as I keep updating).

A lot of blog sites say starting a blog is easy and yet I find starting a blog to be difficult. Other blogsites kind of claims your content and makes you pay for hosting (not my style), so I tried to find something free and easy.
I found Github can host pages for free and uses Jekyll, but you had to install some ruby and then import Jekyll. So I found Jekyll-now on Github to fork over and start posting immediately. I also watched a youtube video that goes through on how to use jekyll-now pretty thoroughly to get me through this.

My next post shall explain what I had learned so far as it has been about a month since I started to learn programming.
